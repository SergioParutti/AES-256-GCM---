# AES-256-GCM шифрование и дешифрование

Локальное настольное приложение для шифрования и дешифрования текста по паролю.

## Возможности

- шифрование текста через `AES-256-GCM`
- дешифрование ранее созданной строки `Base64`
- автоматическая генерация служебных параметров
- простой графический интерфейс на русском языке
- запуск как в GUI, так и через командную строку

## Технологии

- `Python 3.13`
- `tkinter`
- `cryptography`
- `PBKDF2-HMAC-SHA256`

## Быстрый старт

Установить зависимость:

```powershell
py -m pip install -r requirements.txt
```

Запустить программу:

```powershell
py aes256_tool.py
```

## Использование

Шифрование:

1. Введите текст в поле `Исходные данные`.
2. Введите пароль в поле `Пароль`.
3. Нажмите `Зашифровать`.
4. Скопируйте строку из поля `Результат`.

Дешифрование:

1. Вставьте строку `Base64` в поле `Исходные данные`.
2. Введите тот же пароль.
3. Нажмите `Расшифровать`.

## Командная строка

Шифрование:

```powershell
py aes256_tool.py encrypt --password "пароль" --text "текст"
```

Дешифрование:

```powershell
py aes256_tool.py decrypt --password "пароль" --data "BASE64"
```

## Тесты

```powershell
py -m unittest test_aes256_tool.py
```

## Структура проекта

- `aes256_tool.py` — основное приложение
- `test_aes256_tool.py` — автоматические тесты
- `requirements.txt` — зависимости
- `ТЗ - AES-256-GCM.md` — техническое задание
- `Спецификация формата шифропакета.md` — точный формат данных
- `Описание проекта.md` — краткое описание проекта
- `Инструкция по восстановлению приложения.md` — инструкция по восстановлению совместимого аналога

## Важно

Приложение не требует ручного ввода `IV`, `nonce`, `salt` и `tag`. Эти параметры создаются и обрабатываются автоматически.
